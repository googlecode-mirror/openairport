OpenAirport.org - Install Directions

The OpenAirport.org system is 100% open-source and may be downloaded by anyone to use, look-over, or share with other airports. The only request is that you reference OpenAirport.org when using the code for any purpose other than running the OpenAirport.org system. Once you have tried out the demo of OpenAirport and want to try to run it localy on your own computer there are 11 steps you will need to follow: (These instructions assume you are installing on a clean machine with no other web hosting or databses installed.)


Install Guidlines
  • Download a copy of Xampp Lite from http://www.apachefriends.org/en/xampp-windows.html#646
  • Install the Xammp program by compying the contents of the ZIP file to a root folder on your harddrive (C:\xampplite)
  • Run setup_xampp.bat by double clicking on the file.
  • Run the Xampp contorl program by double clicking on the xampp-control.exe file.
  • In the Xampp control start the Apache and MySQL Services by clicking the 'Start' button to the right of the option.
  • Open your web browser and type 'localhost' into the address bar of the browser.
  • Follow the prompts and links to create a password for the MySQL database and PHPMyAdmin.
  • Open PHPMyAdmin, and import the OpenAirport database SQL statement into your MYSQL.
  • Copy the OpenAirport.org code from the Repository to your xammp htdocs folder. (c:\xampplite\htdocs).
  • Open your web browser and type 'localhost' into the address bar of the browser.
  • If everything worked correctly you should see the OpenAirport.orf welcome screen.

Other Considerations
  • The Current Repository of code can be found at Source Forge
  • To get a copy of the OpenAirport.org SQL Statement to import into PHPMyAdmin please contact me and I will email you a copy of the most recent version.
